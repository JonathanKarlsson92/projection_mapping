--------------To run the project-----------------

1: Open a terminal				Ctrl + Alt + T
2: Go to catkin enviroment			cd Path/to/enviroment...
3: Run Rosbridge on port 9090			$roslaunch rosbridge_server rosbridge_websocket.launch
4: Run the project launch file			$roslaunch projection_mapping projection_mapping.launch
5: Open the HTML file proj_mapp_bridge		Open file in project folder
